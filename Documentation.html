Кнопки и их значения:

Кнопка "Добавить текст"
позволяет добавить текст с нужными параметрами.
Кнопка "Добавить кнопку"
позволяет добавить кнопку с нужными параметрами.
Кнопка "Добавить подтекст"
позволяет добавить подтекст с нужными параметрами.
Кнопка "Добавить изображение"
позволяет добавить изображение с нужными параметрами.
Кнопка "Очистить"
позволяет очистисть workspace.
Кнопка "Удалить объект"
позволяет удалить нужный объект.
Кнопка "Сохранить HTML"
позволяет сохранить всё созданное в формате HTML.
Кнопка "Закрыть редактор"
позволяет закрыть редактор.
